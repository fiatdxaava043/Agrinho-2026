# 🌱 Agrinho 2026 — Agro Forte, Futuro Sustentável

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o tema:

> **“Agrinho 2026: Agro Forte, Futuro Sustentável — Equilíbrio entre Produção e Meio Ambiente.”**

O objetivo do site é conscientizar sobre a importância do agronegócio sustentável, mostrando como é possível aumentar a produção agrícola sem prejudicar o meio ambiente.

O projeto apresenta informações sobre sustentabilidade, preservação ambiental e inovação no campo, utilizando tecnologias simples da web como **HTML, CSS e JavaScript**.

---

# 🚜 Objetivos do Site

* Valorizar o agronegócio sustentável;
* Mostrar a importância da preservação ambiental;
* Incentivar práticas agrícolas responsáveis;
* Demonstrar equilíbrio entre produção e natureza;
* Desenvolver habilidades em programação web.

---

# 🛠️ Tecnologias Utilizadas

# 🌍 Sustentabilidade no Agro

O agronegócio é fundamental para a economia e para a alimentação da população. Porém, é necessário produzir de forma consciente, utilizando técnicas sustentáveis que preservem:

* o solo;
* a água;
* as florestas;
* a biodiversidade.

Com tecnologia e responsabilidade ambiental, é possível garantir um futuro melhor para as próximas gerações.
